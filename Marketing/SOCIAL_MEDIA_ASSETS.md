## Image Size Requirements

### Twitter/X
- **Timeline Post:** 1200 x 675 px (16:9 aspect ratio)
- **Header Image:** 1500 x 500 px
- **Profile Picture:** 400 x 400 px (displays as circle)

### LinkedIn
- **Feed Post:** 1200 x 627 px (1.91:1 aspect ratio)
- **Company Banner:** 1128 x 191 px
- **Profile Picture:** 400 x 400 px

### Facebook
- **Feed Post:** 1200 x 630 px (1.91:1 aspect ratio)
- **Cover Photo:** 820 x 312 px
- **Profile Picture:** 180 x 180 px (displays as circle)

### Instagram
- **Feed Post (Square):** 1080 x 1080 px
- **Feed Post (Portrait):** 1080 x 1350 px
- **Story:** 1080 x 1920 px
- **Profile Picture:** 320 x 320 px

## Brand Guidelines for Social Media

### Color Palette

**Primary:**
- Orange: #FF9500
- Blue: #007AFF
- Gradient: 135¬∞ from orange to blue

**Secondary:**
- Dark Text: #1d1d1f
- Light Text: #6e6e73
- Background: #f5f5f7
- White: #ffffff

### Typography

**Headings:** SF Pro Display Bold/Heavy
**Body:** SF Pro Text Regular/Medium
**Fallback:** -apple-system, system-ui

### Logo Usage

- Minimum size: 120px width
- Clear space: 20px on all sides
- Don't stretch, rotate, or alter colors
- Use gradient version on light backgrounds
- Use white version on dark backgrounds

- [ ] Announce coming soon

- [ ] Post launch announcement at 9 AM
