# App Store Preview Video Script & Storyboard

## Video Specifications

**Duration:** 30 seconds (Apple's recommended length)
**Format:** MOV or MP4
**Resolution:** 1920 x 1080 px (16:9 aspect ratio)
**Frame Rate:** 30 fps minimum
**Codec:** H.264, High Profile
**Audio:** AAC, 256 kbps
**File Size:** Under 500 MB
**Captions:** Required (many users watch without sound)

## Production Guide

### Pre-Production

**1. Storyboard Creation**
- Sketch each scene
- Mark timing for each transition
- Note animation requirements
- Plan camera angles

**2. Asset Preparation**
- Create sample data in app
- Prepare demo photos
- Generate example PDFs
- Set up realistic job scenarios

**3. Script Recording**
- Hire professional voiceover artist (or use Text-to-Speech)
- Recommended voices: Clear, friendly, professional
- Record multiple takes
- Edit for natural pacing

**4. Music Selection**
- Use royalty-free music from:
  - Epidemic Sound
  - Artlist
  - AudioJungle
  - Apple Logic Pro loops
- Match energy to video sections
- Ensure proper licensing

---

### Production

**1. Screen Recording Setup**

**Software:**
- **ScreenFlow** (Mac) - $169
- **Camtasia** (Mac/Windows) - $299
- **QuickTime** (Free, basic)

**Settings:**
- Resolution: 1920x1080 minimum
- Frame rate: 60 fps (smooth down to 30 fps in edit)
- Cursor highlight: Optional (can add in post)
- Audio recording: Off (add VO in post)

**Recording Tips:**
- Clean desktop
- Hide menu bar items
- Full screen app recording
- Slow, deliberate movements
- Multiple takes for each action

**2. Filming Live Elements (if applicable)**

**Equipment:**
- DSLR or iPhone 13+ in 4K
- Tripod
- Good lighting (softbox or natural)
- Clean background

**Shots Needed:**
- Contractor at desk (opening hook)
- Hands using iPhone camera
- Contractor smiling (success)
- Over-shoulder typing

**3. Animation Creation**

**Software:**
- **Motion** (Mac) - $50
- **After Effects** (Adobe) - Subscription
- **Keynote** (Mac, Free) - Simple animations

**Animations Needed:**
- Logo reveal
- Text overlays fade in/out
- Number count-ups
- Progress bar fills
- Checkmarks
- Celebration elements

---

### Post-Production

**1. Video Editing**

**Software:**
- **Final Cut Pro** (Mac) - $299
- **DaVinci Resolve** (Free or $295)
- **Adobe Premiere Pro** - Subscription

**Timeline Structure:**
```
Track 1: Main video footage
Track 2: Overlay graphics/text
Track 3: Transitions
Track 4: Music
Track 5: Voiceover
Track 6: Sound effects
```

**2. Color Grading**
- Ensure consistent colors
- Match app colors (orange, blue)
- Brighten slightly for visibility
- Avoid oversaturation

**3. Audio Mixing**
- Voiceover: -6dB to -3dB (clear and loud)
- Music: -20dB to -18dB (background)
- Sound effects: -12dB to -10dB (noticeable)
- Duck music under voiceover
- Fade music in/out smoothly

**4. Captions**
- Add burned-in captions (always visible)
- Use large, readable font (60pt+)
- High contrast (white text, black outline)
- Position: Lower third or top
- Follow voiceover word-for-word

**5. Export Settings**
```
Format: H.264
Resolution: 1920x1080
Frame Rate: 30 fps
Bitrate: 10-15 Mbps
Audio: AAC 256 kbps
```

## Testing & Optimization

### Before Publishing

**1. Quality Check**
- Watch at full volume and muted
- Test on different devices (Mac, iPhone, iPad)
- Verify captions are readable
- Check for typos in text overlays
- Ensure all animations are smooth

**2. User Testing**
- Show to 5-10 contractors
- Ask: "What does this app do?"
- Ask: "Would you download it?"
- Note confusion points
- Iterate based on feedback

**3. A/B Testing (After Launch)**
- Create 2-3 variations
- Test different hooks
- Test different feature orders
- Measure conversion rates
- Optimize for best performer

## Success Metrics

Track these metrics after publishing:

**App Store Connect Analytics:**
- Video view rate (% of page visitors who watch)
- Video completion rate (% who watch to end)
- Conversion rate (video view → download)

**Goals:**
- 60%+ view rate
- 50%+ completion rate
- 15%+ conversion rate (video view to download)

If metrics are below goals:
- Test different thumbnails
- Shorten video
- Improve hook
- Add more value in first 3 seconds
