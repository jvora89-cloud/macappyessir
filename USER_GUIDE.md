## üßô Creating Estimates

### Using the 5-Step Wizard

**Step 1: Choose Method**
- **AI Camera**: Use camera to estimate costs (experimental)
- **Use Template**: Start from previous estimate
- **Start Fresh**: Create from scratch

**Step 2: Client Information**
- Client name (required)
- Address
- Phone number
- Email address
- **Tip**: Auto-complete suggests existing clients as you type

**Step 3: Job Details**
- Select job type (Plumbing, Electrical, HVAC, etc.)
- Job description
- Start date
- Estimated completion date

**Step 4: Pricing**
- Enter estimated cost
- Select pricing type (Fixed, Hourly, Per Item)
- Add line items (optional)
- **Tip**: Line items help clients understand the breakdown

**Step 5: Review**
- Review all information
- Click "Create Estimate" to save
- Or go back to edit any step

### Draft Estimates

Your estimate is automatically saved as a draft. You can:
- Resume drafts from Step 2
- Delete old drafts
- Have multiple drafts in progress

## üí∞ Payment Tracking

### Setting Up Payment Schedule

**Option 1: Percentage-Based**
- Define payment milestones
- Example: 33% deposit, 33% midpoint, 34% final
- App auto-calculates amounts

**Option 2: Milestone-Based**
- Custom milestone amounts
- Example: $5k deposit, $7k after framing, $3k final
- More flexible for complex jobs

### Recording Payments

1. Navigate to job details
2. Click "Add Payment" in payment section
3. Fill in:
   - Amount
   - Payment method (Cash, Check, Card, Bank Transfer, etc.)
   - Date received
   - Reference number (optional)
   - Notes (optional)
4. Click "Save"

### Payment Receipt

Generate receipts for clients:
1. Click payment in list
2. Click "Generate Receipt"
3. Preview the receipt
4. Export as PDF or print

### Payment Analytics

View payment insights:
- **Total Received**: Sum of all payments
- **Outstanding Balance**: Remaining amount due
- **Payment Method Breakdown**: Pie chart showing cash vs card vs check
- **Payment Timeline**: Line chart showing payments over time

1. Quit the app
2. Copy the folder above to external drive
3. Store backup safely

1. Quit the app
2. Copy backed-up folder to original location
3. Restart the app
4. Your data is restored

### Exporting Data

Export to CSV:
1. Go to Jobs list
2. Click "Export" button
3. Choose "CSV"
4. Open in Excel/Numbers

## üÜò Common Issues

### "Permission Denied" When Taking Photos

**Solution:**
1. Open System Settings
2. Go to Privacy & Security ‚Üí Camera
3. Enable Lakshami Contractors
4. Restart the app

### PDF Won't Export

**Solution:**
1. Try exporting to Desktop
2. Check disk space
3. Verify you have write permission to destination folder

### App Crashes on Launch

**Solution:**
1. Restart your Mac
2. Update to latest macOS
3. Reinstall the app
4. Contact support if persists

### Data Not Saving

**Solution:**
1. Check disk space
2. Verify app has permission to write files
3. Check Console.app for errors
4. Contact support with error messages
