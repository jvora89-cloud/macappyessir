## üêõ Troubleshooting

**Q: The app won't open. What should I do?**
A:
1. Make sure you're running macOS 13+
2. Try restarting your Mac
3. Reinstall the app
4. Email jvora89@yahoo.com if the problem persists

**Q: Camera isn't working**
A:
1. Go to System Settings ‚Üí Privacy & Security ‚Üí Camera
2. Make sure Lakshami Contractors is checked
3. Restart the app

**Q: PDFs aren't exporting**
A:
1. Check that you have write permission to the destination folder
2. Try exporting to Desktop
3. Make sure you have disk space available

**Q: My data disappeared**
A:
Data is never deleted automatically. Check:
1. Are you using the correct Mac user account?
2. Check ~/Library/Application Support/Lakshami Contractors
3. Contact support if you need help recovering data

**Q: The app is slow/laggy**
A:
1. Close other apps to free up RAM
2. Restart the app
3. Check if you have many large photos (compress them)
4. Update to the latest version
