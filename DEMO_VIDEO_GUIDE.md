## üìπ Recording Setup

### Software Needed
- **Screen Recording**: QuickTime Player (built-in) or ScreenFlow
- **Video Editing**: iMovie (free) or Final Cut Pro
- **Audio**: GarageBand (voiceover) or Descript
- **Graphics**: Keynote (for title cards) or Canva

### Recording Checklist

**Before Recording:**
- [ ] Load sample data in app (use Developer Tools)
- [ ] Set Mac resolution to 1920x1080
- [ ] Close all other apps (clean desktop)
- [ ] Disable notifications (Focus mode)
- [ ] Clear browser history/tabs
- [ ] Set mouse cursor to be visible in recordings
- [ ] Test audio levels

**During Recording:**
- [ ] Record each section separately (easier to edit)
- [ ] Use smooth, deliberate cursor movements
- [ ] Pause 2 seconds between actions
- [ ] Capture at 60fps for smooth animations
- [ ] Leave 5 seconds at start/end of each clip

**After Recording:**
- [ ] Trim clips to exact timing
- [ ] Add transitions (0.5s cross-dissolve)
- [ ] Overlay keyboard shortcuts graphics
- [ ] Add text overlays
- [ ] Color grade for consistency
- [ ] Add background music (royalty-free)
- [ ] Record and sync voiceover
- [ ] Add sound effects (subtle)
- [ ] Export at 1080p, H.264, 30fps

## üí° Pro Tips

1. **Show, Don't Tell**: Let the app speak for itself with smooth interactions
2. **Pacing**: Keep it moving - no shot longer than 10 seconds
3. **Real Data**: Use realistic sample data, not obviously fake content
4. **Cursor**: Make cursor movements smooth and intentional, never erratic
5. **Audio**: Invest in good voiceover - it makes or breaks the video
6. **Subtitles**: Add captions for accessibility and silent viewing
7. **Hook**: First 5 seconds must grab attention
8. **CTA**: Clear call-to-action at the end with URL visible
